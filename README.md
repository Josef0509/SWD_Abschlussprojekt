# SWD_Abschlussprojekt

Abschlussprojekt für Softwaredesign

## Installationsanleitung :

### 1 Setting up VENV
```bash
python -m venv venv
```

### 2 Activating the VENV
```bash
.\venv\Scripts\activate
```

### 3 Install packages
```bash
pip install streamlit, st_pages, matplotlib
```
